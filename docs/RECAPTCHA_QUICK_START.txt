═══════════════════════════════════════════════════════════════════════
🌐 Visit: https://www.google.com/recaptcha/admin/create

1. Log in with Google account
2. Fill in the form:

   │ Label:           GrowLokal                                       │
   │ reCAPTCHA type:  ✓ reCAPTCHA v3                                 │
   │ Domains:         localhost                                       │
   │                  growlokal.com (your domain)                     │
   └─────────────────────────────────────────────────────────────────┘
3. Submit and copy your keys:
   📋 Site Key:   6LcXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX (public)
   🔑 Secret Key: 6LcYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY (private)


STEP 2: UPDATE .env.local
═══════════════════════════════════════════════════════════════════════
📝 File: .env.local (in project root)

Add these lines:
┌─────────────────────────────────────────────────────────────────────┐
│ # reCAPTCHA v3 Configuration                                        │
│ NEXT_PUBLIC_RECAPTCHA_SITE_KEY=your_site_key_here                  │
│ RECAPTCHA_SECRET_KEY=your_secret_key_here                          │
│                                                                      │
│ # Optional: Show dev links                                          │
│ NEXT_PUBLIC_SHOW_DEV_LINKS=true                                    │
└─────────────────────────────────────────────────────────────────────┘

⚠️  IMPORTANT: Replace "your_site_key_here" and "your_secret_key_here"
              with your actual keys from Step 1!

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════
❌ "reCAPTCHA site key not configured"
   → Check .env.local has NEXT_PUBLIC_RECAPTCHA_SITE_KEY
   → Restart dev server

❌ reCAPTCHA badge not showing
   → Open browser console (F12)
   → Check for JavaScript errors
   → Verify site key is correct

❌ "Invalid site key"
   → Make sure you copied the Site Key (not Secret Key)
   → Check for extra spaces in the key
   → Verify key is for reCAPTCHA v3 (not v2)

❌ Verification failing on server
   → Check RECAPTCHA_SECRET_KEY in .env.local
   → Ensure "localhost" is added in Google admin console
   → Check internet connection
